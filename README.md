# Pilotoapi
Pilotoapi is a simple API to manage products from mysql database.

## Installation
Install docker and docker-compose

## Usage
```bash
make external
make run
```

## Endpoints
```bash
### GET /api/product
### GET /api/product/{id}
### POST /api/product
### PUT /api/product/{id}
### DELETE /api/product/{id}
```
